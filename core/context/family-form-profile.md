## Form-Filling Rules

- Use `preferred_name` only when the form is clearly informal or already uses that variant.
- Use `full_name` for legal waivers, school forms, insurance forms, and government forms.
- Use this file as the primary source for passport and driver's license values.
- Use the referenced source documents only if a field is missing here or needs verification.
- Do not infer or fabricate missing DOBs, ID numbers, passport numbers, or signatures.
